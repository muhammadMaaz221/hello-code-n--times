# 👋 Hello World N Times (C++)

## 📌 Project Overview

This is a simple C++ program that prints **"Hello World"** multiple times based on user input.

---

## 🎯 Objective

* To understand loops in C++
* To practice user input and output
* To learn how repetition works using loops

---

## 💻 How It Works

The program:

1. Takes a number `n` from the user
2. Uses a loop to repeat the process
3. Prints **"Hello World"** `n` times

---

## 🧠 Logic Used

* A `for` loop runs from 1 to `n`
* Each iteration prints the message

---

## ▶️ Example

Input:

```id="input123"
5
```

Output:

```id="output123"
Hello World
Hello World
Hello World
Hello World
Hello World
```

---

## 🚀 How to Run

1. Copy the code into a `.cpp` file
2. Compile using a C++ compiler (e.g., g++)
3. Run the program
4. Enter a number
